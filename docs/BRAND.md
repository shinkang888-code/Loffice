# Loffice 브랜드 가이드

## 로고

![Loffice Logo](../assets/logo.png)

**파일**: `assets/logo.png`

### 디자인 설명

- **외곽 프레임**: 브러시드 실버 메탈, 둥근 모서리 — 프리미엄 소프트웨어 아이콘 스타일
- **L 자형 요소**: 틸(teal) 컬러 브러시드 메탈 — "Loffice"의 L을 상징
- **중앙 골드 스퀘어**: 웜 골드/bronze 버튼 — 문서·오피스 핵심 기능 상징
- **배경**: 매트 화이트 — 깔끔한 문서 작업 환경
- **스타일**: 3D 스큐어모피즘, 산업적·전문적

### 사용 규칙

- 최소 크기: 32×32px (favicon), 권장 128×128px 이상
- 로고 주변 여백: 로고 높이의 25% 이상
- 배경이 어두울 때: 원본 로고 사용 (자체 화이트 배경 포함)
- 변형 금지: 비율 변경, 색상 단독 변경, 그림자 추가

## 제품명

- 공식 명칭: **Loffice** (L 대문자, office 소문자)
- 한글: 로피스
- 도메인 후보: `loffice.app`, `loffice.io`
- 태그라인: *"LibreOffice, Reimagined for the Web"*
